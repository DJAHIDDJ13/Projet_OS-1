# File Manager
file manager that uses a simulated partition (UNIX file), OS Project

## getting started
### installing
To install this project:
To download the project
```
	git clone https://www.github.com/ishak96/PROJET_OS
```
Then go to the directory
```
	cd PROJET_OS
```
compile the project
```
make
```
To run the project
```
	./bin/fmanag [-d] [dir]
```
